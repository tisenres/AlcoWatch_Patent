# WESAD Dataset Download Instructions

1. Register and download from: https://uni-siegen.sciebo.de/s/HGdUkoNlW1Ql4x866
2. Extract the downloaded archive
3. Place the extracted data in: stress_detection/data/WESAD/

Expected structure after extraction:
```
stress_detection/data/WESAD/
├── S2/S2.pkl
├── S3/S3.pkl
├── S4/S4.pkl
...
└── S17/S17.pkl
```

Note: S1 was a pilot study and is excluded. S12 was not collected by the dataset authors and is absent from the archive — this is expected. The archive contains 15 subjects total (S2–S11, S13–S17).
